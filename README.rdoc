== haplo.io - Project Management Tool

This is a multi-tenant project management tool written in Rails and Angular, its frontend is SPA application. It was done for educational purposes.

It has features like:
- Ticket tracking
- Time tracking
- Milestones
- Discussions which can be done over different models

The Angular application can be found at `app/assets/javascripts/main.js.coffee` and its files in the `app/assets/javascripts/angular`.
